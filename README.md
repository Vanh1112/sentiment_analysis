## Docker
```
docker build -t tinhte-sentiment:latest .
docker run -dt -v $(pwd):/app/ -p 5000:5000 --env-file docker/.env --name sentiment sentiment_analysis:latest
```

```
Train: python3 -m app.train
App: python3 -m app.run
```
